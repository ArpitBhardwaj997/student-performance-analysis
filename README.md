# student-performance-analysis
End-to-end data analysis project using SQL, Python (pandas), and matplotlib to analyze student performance.

# 📊 Student Performance Analysis

## 🚀 Overview
This project performs an end-to-end data analysis on student performance using SQL and Python. It demonstrates how raw data is extracted, processed, analyzed, and visualized to generate meaningful insights.

---

## 🛠 Tech Stack
- Python (pandas, matplotlib)
- PostgreSQL (SQL)
- CSV / Excel

---

## 🔄 Workflow
1. Data stored in PostgreSQL database
2. Extracted data using SQL queries
3. Data cleaned and processed using pandas
4. Created new feature (Average Marks)
5. Visualized data using matplotlib
6. Exported processed data to CSV file
